TECHNICAL SPECIFICATION: HVNC PLATFORM

1. PROJECT OVERVIEW

A high-performance remote access platform allowing administrators to manage devices and shifts, and authorized users to access remote Chrome browser sessions via a web-based portal.

2. DESIGN SYSTEM (CHARCOAL & ORANGE)

  • Primary Background: linear-gradient(135deg, #333333, #F6921E)
  • Surface/Card: #2B2B2B
  • Primary Actions: #333333 base, #F6921E accents

3. CORE MODULES

Admin Portal

Dashboard, Device Management, User Assignment, and Shift Scheduling.

User Portal

Access Code Entry, Connection Sequence, and Hidden Chrome Session.

4. BACKEND REQUIREMENTS

  • Auth Service: JWT + 2FA TOTP.
  • Device Service: WebSocket heartbeats for real-time status.
  • Session Service: Secure one-time-use access code logic.
  • Streaming: E2EE remote desktop protocol for Chrome sessions.
