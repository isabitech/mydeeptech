Background Colors
Primary Gradient Background
For the overall background, creating a clean, professional vibe:

css
Copy code
background: linear-gradient(135deg, #333333, #F6921E);
#333333: Primary dark gray for a solid, neutral base.
#F6921E: Vibrant orange for a warm contrast.
Secondary Section Background
For highlighting specific sections like features:

css
Copy code

#333333: The base color.

Accent Background
For cards, modals, or containers:

css

Button Colors
Primary Button
Designed for primary actions:

css
Copy code
background: #333333;
color: #FFFFFF;

&:hover {
  background: #292929;
}
#333333: Strong, grounded primary color.
#292929: Slightly darker hover state for tactile feedback.
#FFFFFF: White text for maximum readability.
